# Pandas Basic to Advanced - Easy Guide

This repository contains a simple Python tutorial script demonstrating basic to intermediate usage of the `pandas` library.

## Contents

- `PANDAS BASIC TO ADVANCED - EASY GUIDE.py` - a walkthrough of common pandas operations, including DataFrame creation, reading CSV files, and data inspection.

## Getting Started

### Prerequisites

- Python 3.7+
- `pandas`

Install pandas with:

```bash
pip install pandas
```

### Run the script

```bash
python "PANDAS BASIC TO ADVANCED - EASY GUIDE.py"
```

## What this script shows

- Creating a DataFrame from a dictionary
- Printing DataFrame contents
- Viewing data with `head()`, `tail()`, `shape`, `columns`, and `info()`
- Placeholder example for reading a CSV file with `pd.read_csv()`

## Notes

This repository is meant for learners who want a quick, easy-to-follow pandas example. The script can be extended with additional pandas operations as needed.
